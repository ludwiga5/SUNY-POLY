# SUNY Polytechnic University Coursework

A collection of coursework and projects from computer science courses at SUNY Polytechnic University.

## Courses

### CS108 - Introduction to Programming
Foundational programming concepts in C, covering:
- Basic syntax and control flow (ch1-ch3)
- Functions and modular design (ch5-ch6)
- Arrays and data structures (ch7)
- String manipulation (ch8)
- Linked lists and recursion (ch9)
- Homework assignments and extra credit exercises

### CS220 - Digital Image Processing
Image processing fundamentals using C:
- **HW1 Section-1**: Black and white image manipulation (PGM format)
- **HW1 Section-2**: Color image processing (PPM format)
- Labs covering C compilation pipeline and review

### CS240 - Object-Oriented Programming
C++ programming and object-oriented design:
- Basic C++ syntax and OOP concepts (Week 1-3)
- Memory allocation and pointers (Week 3-4)
- Dynamic arrays and data structures (Week 4)
- Classes and object design (Week 5-6)
- Linked lists and string handling (Week 6)
